# AzSeo вҖ” Firefox Version
This is a Firefox port of the [Original AzSeo]([AzSeo](https://github.com/TheGreatAzizi/AzSeo--Real-time-SEO-analysis-ChromeExtension)) by @TheGreatAzizi.

**Adapted for Firefox by:** @alexantSWE

## вңҰ What is AzSeo?

**AzSeo** is a zero-dependency Chromium browser extension that performs instant, in-depth SEO audits on any webpage you visit. It reads the page's DOM locally вҖ” no data ever leaves your browser вҖ” and returns a detailed score across 15 key SEO factors, ranked tips, and a full feature-flag breakdown.

Built for developers, SEO specialists, and anyone who cares about search visibility.

---

## вңҰ Features

- **15-factor SEO audit** вҖ” title, meta, headings, HTTPS, images, canonical, OG tags, schema & more
- **100-point scoring system** вҖ” with letter grade (A+ вҶ’ F) and animated score dial
- **3 analysis panels** вҖ” Checks (per-item breakdown), Overview (stats & flags), Tips (prioritized actions)
- **Priority-ranked recommendations** вҖ” Critical вҶ’ High вҶ’ Medium вҶ’ Low, with impact bars
- **Feature flag grid** вҖ” instant view of 8 key technical SEO flags
- **Zero dependencies** вҖ” pure vanilla JS, no frameworks, no npm, no bundler
- **Works on any page** вҖ” blogs, e-commerce, SPAs, news sites, landing pages
- **Private by design** вҖ” all analysis happens in your browser, nothing is sent anywhere

## рҹҡҖ Installation

AzSeo is not yet on the Chrome Web Store вҖ” load it manually in any Chromium browser (Chrome, Edge, Brave, Arc, Opera).

**1 вҖ” Clone or download**

```bash
git clone https://github.com/TheGreatAzizi/AzSeo--Real-time-SEO-analysis-ChromeExtension.git
```

Or grab the latest ZIP from [Releases](https://github.com/TheGreatAzizi/AzSeo--Real-time-SEO-analysis-ChromeExtension/releases).

**2 вҖ” Open Extensions**

```
chrome://extensions
```

**3 вҖ” Enable Developer Mode**

Toggle **Developer mode** on (top-right corner).

**4 вҖ” Load Unpacked**

Click **Load unpacked** вҶ’ select the cloned/extracted folder.

**5 вҖ” Pin & Go**

Click the puzzle icon in your toolbar вҶ’ pin **AzSeo** вҶ’ open it on any page.

AzSeo injects `content.js` into the active tab. It reads the DOM вҖ” headings, meta tags, links, images, structured data вҖ” and sends that snapshot to the popup. The popup's scoring engine evaluates all 15 factors, generates ranked tips, and renders everything locally. No network calls are made at any point.

## рҹ”’ Permissions

AzSeo requests the absolute minimum:

| Permission | Purpose |
|---|---|
| `activeTab` | Access the URL and tab context of the current page |
| `scripting` | Inject `content.js` to read the page DOM |

No storage access. No host permissions. No external requests. Your browsing data stays yours.
